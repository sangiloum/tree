# IBS DIMAG Academic Family Tree

Interactive visualization of advisor/student genealogy for members of the
[IBS Discrete Mathematics Group (DIMAG)](https://dimag.ibs.re.kr).

Data sourced from the [Mathematics Genealogy Project](https://www.mathgenealogy.org).

---

## Quick Start

```bash
# 1. Install Python dependencies
cd scripts
pip install -r requirements.txt

# 2. Run the scraper (takes ~10–30 min depending on tree size)
python scrape_genealogy.py

# 3. Serve locally
cd ..
python -m http.server 8000
# Open http://localhost:8000
```

## Updating Member Data

Edit `data/members.json` to add/remove DIMAG members. Each entry needs:
- `mgp_id` — their Mathematics Genealogy Project ID (look up at mathgenealogy.org)
  or `null` if they don't have an MGP entry
- `name` — full name as it appears on MGP
- `role` — their DIMAG role

After updating `members.json`, re-run the scraper to regenerate `genealogy.json`.

---

## Scraper Configuration

Edit the constants at the top of `scripts/scrape_genealogy.py`:

| Variable | Default | Description |
|---|---|---|
| `ANCESTOR_DEPTH` | `None` | Max generations upward (`None` = unlimited) |
| `DESCENDANT_DEPTH` | `3` | Max generations downward from seeds |
| `SLEEP_BETWEEN_REQUESTS` | `1.5` | Seconds between requests (be polite!) |

## Features

- **Dark cosmic theme** — gold for current DIMAG, teal for former DIMAG
- **Force-directed layout** — stratified by PhD year (older = higher)
- **Progressive disclosure** — depth slider controls visible ancestor depth
- **Hover tooltips** — name, institution, PhD year, dissertation
- **Click sidebar** — full details, advisor/student lists, MGP link
- **Double-click ancestor** — highlights path down to DIMAG members
- **Search** — fuzzy name search with camera animation
- **Zoom/pan** — SVG canvas with LOD (labels/deep nodes hidden at low zoom)
- **Mobile fallback** — table view on narrow screens
